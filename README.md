## Program Structure

Classes name are the same just without spaces

#### Login and Signup
- Front End
  - Sign Up
  - Log In
- Back End
  - Fetch Account Data
  - Post Account Data

#### Matching System
- Front End
  - Matching Layout
    - Search Bar
    - Matching Card 
- Back End
  - Fetched Card Data
  - Search Functionality

#### Bulletin System
- Front End
  - Bulletin Layout
  - Bulletin Create Button
  - Bulletin Card
- Card Popup
- Back End
  - Fetch Data for card
  - Post Data when creating card
